Palette viewer




What's it for

The palette viewer is for people who do graphics hacking and want to know the exact details of how the game colour system (PLAYPAL and COLORMAP) works. It lets you know, for example :

  • how colour numbers are mapped to RGB values,
  • how colours change with brightness,
  • how many colour numbers are mapped to by a given colour number,
  • how many colour numbers a given colour number maps to.
